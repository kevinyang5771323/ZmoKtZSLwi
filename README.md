# 停车场管理系统 java46

## 项目概述

停车场管理系统是一个基于B/S架构的web网站，采用前后端分离的模式开发。该系统致力于提供便捷的停车服务与管理功能。

## 技术栈

- **架构**: B/S、MVC
- **开发环境**：
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql 8.0
- **技术栈**：
  - Java
  - Mysql
  - SpringBoot
  - Mybatis-Plus
  - Vue

## 功能模块

### 用户角色

#### 普通用户

- 个人信息管理
- 预定停车位
- 缴费信息管理

#### 管理员用户

- 用户信息管理
- 车辆信息管理
- 停泊车辆管理
- 车辆进出管理
